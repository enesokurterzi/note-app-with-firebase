# Note App With Firebase

Description
-------------

<p>
In this project, you can create or edit notes and the app requires login. You can create account or reset password.
  
| Pages in the app |
| --------- |
|  Main     |
|  Create Account    | 
|  Login     |
|  Note Details  |
|  Splash  |
  
## Libraries And Features
  
  - ViewBinding <p>
    
  - Firebase <p>
   
  - Firebase - auth <p>  
  
  - Firestore <p>
